# adamclark64/homebrew-tap

Homebrew formulas for [adamclark64](https://github.com/adamclark64)'s tools.

## Usage

```sh
brew tap adamclark64/tap
brew install ccmon
# or in one shot:
brew install adamclark64/tap/ccmon
```

## Formulas

- [`ccmon`](https://github.com/adamclark64/ccmon) — TUI monitor for Claude Code agents & subagents

> Formulas in this tap are auto-updated by each upstream project's release
> workflow. Do not hand-edit `Formula/*.rb`.
